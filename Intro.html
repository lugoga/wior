1 Introduction

Binning is a way to group a number of more or less continuous values into a smaller number of “bins”. For example, if you have data about a group of people, you might want to arrange their ages into a smaller number of age intervals.

Statistical data binning is a way to group numbers of more or less continuous values into a smaller number of “bins”. For example, if you have data about a group of people, you might want to arrange their ages into a smaller number of age intervals (for example, grouping every five years together). It can also be used in multivariate statistics, binning in several dimensions at once.

1.1 Packages

Our wior package has a binning_points function that is capable of binning and computed weighted means of the continuous values at 25th , 50th and 90th quantiles. This allows the analyst to explore the distribution of the weighted values in each bin at the quantile of choice. To work with the wior package, we need to load other packages in the session using either library or require function.

2 Converting simple feature to tibble

Sometimes you may wish to work in tibble. However, the only dataset you have is simple feature. Though the simple feature contain attributes information that describe or characterize the feature. But when you may want to only remain with the tibble with their geographical position and drop the geometry attributes in simple feature. The wior package has two functions that can assist you with this

2.1 Change point to tibble

As the name of point_tb imply, this function change point feature into a tible dataset. let us see how
